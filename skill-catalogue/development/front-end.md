# 前端开发技能 (Front-End Development)

> 适用于 UI 设计、组件架构、原型设计及图表绘制等前端场景。

## 构建工具与设计

| Skill 名称 | 调用方式 | 触发条件/适用场景 |
| :--- | :--- | :--- |
| **ui-ux-pro-max** | `Skill(name="ui-ux-pro-max")` | UI/UX 设计智能支持，50 种风格、21 配色板、50 字体配对，支持 React/Vue/Svelte/Flutter 等多框架。 |
| **drawio** | `Skill(name="drawio")` | Draw.io 图表创建/编辑/复制，支持架构图、流程图、网络拓扑、UML、ER 图、学术论文插图等。 |
| **excalidraw-diagram-generator** | `Skill(name="excalidraw-diagram-generator")` | Excalidraw 图表生成，从自然语言描述生成流程图、架构图、思维导图、时序图、ER 图等 9 种图表，输出 .excalidraw JSON 文件。 |
| **design-studio** | `Skill(name="design-studio")` | HTML 高保真原型、交互 Demo、幻灯片、动画、设计变体的一体化设计能力，含 20 种设计哲学、品牌资产协议、MP4/GIF 视频导出。 |
